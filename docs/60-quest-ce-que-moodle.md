# Ⓜ️ Qu'est-ce que Moodle

- [ ] Moodle est un système de gestion de l’apprentissage
- [ ] Le système permet de présenter des cours et leurs contenus
- [ ] Enfin, il permet d’évaluer les étudiant.es à la fin des cours.
